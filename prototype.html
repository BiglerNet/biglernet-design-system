Using Tokens

For back-end devs: tokens are just CSS custom properties (variables). Copy this into your project and go.

CSS — paste into your :root
/* In your global CSS file: */
:root {
  /* Backgrounds */
  --bg:      oklch(99% 0.002 240);
  --surface: oklch(100% 0 0);

  /* Text */
  --fg:      oklch(18% 0.012 250);
  --muted:   oklch(54% 0.012 250);

  /* Accent — BiglerNet Red */
  --accent-primary:   oklch(52% 0.21 23);
  --accent-secondary: oklch(42% 0.18 23);
  --accent-light:     oklch(90% 0.06 23);

  /* Border & shadow */
  --border:  oklch(90% 0.006 250);
  --radius-md: 8px;
  --shadow-sm: 0 1px 2px rgba(0,0,0,0.06);

  /* Semantic */
  --success: oklch(65% 0.14 150);
  --danger:  oklch(58% 0.18 15);
}
Dark mode — override in a .theme-dark class
.theme-dark {
  --bg:      oklch(14% 0.01 250);
  --surface: oklch(19% 0.01 250);
  --fg:      oklch(94% 0.01 240);
  --muted:   oklch(56% 0.01 240);
  --border:  oklch(28% 0.01 240);
}
How to use a token in your components
/* In any component CSS: */
.button-primary {
  background: var(--accent-primary);
  color: #fff;
  border-radius: var(--radius-md);
  padding: 8px 24px;
}

.card {
  background: var(--surface);
  border: 1px solid var(--border);
  border-radius: var(--radius-lg);
  padding: 24px;
}

h1 {
  font-family: 'Inter', system-ui, sans-serif;
  font-size: 32px;
  font-weight: 600;
  color: var(--fg);
  letter-spacing: -0.02em;
}
